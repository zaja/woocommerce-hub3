== Description ==

**PDF417 - 2D barcode generator for Bank Payment(basc)**

**Features**

* Create HUB3 - 2D barcode regarding specification of Croatian Banking Association

**Translations**

* English
* Croatian

== Installation ==

1. Upload the `hub3-barcode` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Woocommerce > Setting > HUB3 2D tab and change setting
